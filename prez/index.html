hello world

The canonical node.js hello world example. 

var http = require('http');

http.createServer(function (request, response) {
    response.writeHead(200, {'Content-Type': 'text/plain'});
    response.end('Hello World\n');
}).listen(8124);

console.log('Server running at http://127.0.0.1:8124/');

events! events! events!

Whether implicit or explicit, understanding callbacks is critical.

implicit - one shot

Passing a function directly to another function as a callback. 

do_something(input, function(err, result) { ... });

explicit - many or single

Using the EventEmitter interface to connect callbacks. 

instance.on('some event', function(result) { ... });

modules

node.js core is intentionally small. 3rd party modules provide all the other "goodies" (this is by design)

modules can refer to both 3rd party modules you install as well as your own files. Both are exposed through the same interface module.exports and require

Understanding modules is crucial to becoming proficient in node.js land. DRY

Take time to read and understand the module docs

Start using npm from day one (today!).

Track your dependencies using a package.json file.
npm init will help you get started

Most importantly: use, create, share great modules!!
